# 登录注册流程图

## 应用启动流程

```
启动页 (SplashPage)
    ↓
检查认证状态
    ↓
┌─────────────────┬─────────────────┐
│   已登录        │    未登录        │
│       ↓         │       ↓         │
│   主页 (Home)   │  登录页 (Login)  │
└─────────────────┴─────────────────┘
```

## 登录流程

```
登录页 (LoginPage)
    ↓
输入用户名和密码
    ↓
表单验证
    ↓
┌─────────────────┬─────────────────┐
│   验证失败      │    验证成功      │
│       ↓         │       ↓         │
│   显示错误      │   调用登录API    │
│   重新输入      │       ↓         │
└─────────────────┘   登录结果      │
                        ↓         │
                ┌─────────────────┬─────────────────┐
                │   登录失败      │    登录成功      │
                │       ↓         │       ↓         │
                │   显示错误      │   跳转到主页    │
                │   重新登录      │   (HomePage)    │
                └─────────────────┴─────────────────┘
```

## 注册流程

```
登录页 (LoginPage)
    ↓
点击"立即注册"
    ↓
注册页 (RegisterPage)
    ↓
填写注册信息
    ↓
表单验证
    ↓
┌─────────────────┬─────────────────┐
│   验证失败      │    验证成功      │
│       ↓         │       ↓         │
│   显示错误      │   调用注册API    │
│   重新填写      │       ↓         │
└─────────────────┘   注册结果      │
                        ↓         │
                ┌─────────────────┬─────────────────┐
                │   注册失败      │    注册成功      │
                │       ↓         │       ↓         │
                │   显示错误      │   返回登录页    │
                │   重新注册      │   (LoginPage)   │
                └─────────────────┴─────────────────┘
```

## 页面跳转关系

```
SplashPage
    ↓
┌─────────────────┬─────────────────┐
│   AppRoutes.home│  AppRoutes.login│
│       ↓         │       ↓         │

│                 │  RegisterPage   │
└─────────────────┴─────────────────┘
```

## 功能特性

### 登录页面功能
- ✅ 用户名输入（3-20字符）
- ✅ 密码输入（6-20字符）
- ✅ 密码可见性切换
- ✅ 记住密码选项
- ✅ 忘记密码功能
- ✅ 快速登录演示
- ✅ 跳转到注册页

- ✅ 用户名输入（3-20字符）

- ✅ 密码输入（6-20字符）
- ✅ 确认密码（必须匹配）
- ✅ 用户协议同意
- ✅ 隐私政策查看
- ✅ 返回登录页

### 通用功能
- ✅ 表单验证

- ✅ 错误提示
- ✅ 成功反馈
- ✅ 页面转场动画
- ✅ 响应式设计

## 测试用例

### 登录测试
1. **正常登录**：用户名 demo，密码 123456
2. **快速登录**：点击"快速登录（演示）"按钮
3. **验证测试**：输入无效格式的用户名或密码
4. **记住密码**：勾选记住密码选项

### 注册测试
1. **正常注册**：填写所有必填信息
2. **验证测试**：测试各种无效输入
3. **协议测试**：不勾选同意条款尝试注册
4. **密码匹配**：输入不匹配的确认密码
